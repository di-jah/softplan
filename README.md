 Description
SoftPlan is a Flutter-based task management application that allows users to create tasks, assign priorities, mark tasks as completed, view task details, and delete tasks.
The app is built without any backend or database; tasks are stored temporarily in memory using `setState()`.

 Features
- Splash / welcome screen
- Add tasks with title, description, and priority
- View tasks in a list
- Mark tasks as completed using a checkbox
- View task details with status (Pending / Completed)
- Delete tasks
- Snackbar feedback for user actions

 Technologies Used
- Flutter
- Dart
- Material UI

 How to Run
1. Run `flutter pub get`
2. Run `flutter run`
